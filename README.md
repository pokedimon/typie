# Тайпи
**Тайпи** - клавиатурный тренажер для детей, которые еще только приучаются к клавиатуре и не могут свободно печатать. Формат представляет собой игру, требующую взаимодействия с клавиатурой и развивающую навыки печатания.

## Пояснительная записка
Идея заключается в том, чтобы создать игру, которая одновременно развивает навыки печатания, учит алфавиту и содержит социальное взаимодействие. Для начала, есть два режима игры - песочница со различными настройками (скорость, буквы, таймер и т.п.), результаты которой учитываются в рейтинге, и уровни, последовательно проходящие по всем буквам русского алфавита и увеличивающиеся в сложности. На сайте можно зарегистрироваться и тем самым получить возможность соревноваться с другими игроками в таблице лидеров, а также изменять данные в профиле и добавлять друзей.  
  
Все шаблоны подстроены под *base.html*, в котором содержится подключение всех скриптов и стилей, шапка и окно входа/регистрации. Игровой процесс написан в *game-logic.js* и возвращает шаблону *game.html* функцию инициализации игры. Таблицу лидеров можно отсортировать и отфильтровать по классу, имени, счету; логика написана в *leaderboard-sorting.js* и также возвращает функцию в *leaderboard.html*.  
Используется база данных с двумя связанными таблицами: *users* и *games*. В *users* содержатся данные о пользователях (логин, пароль, класс, имя и т.п.), а в *games* - счет и параметры определенной игры в песочнице, причем каждая из игр соответствует какому-либо из пользователей. При окончании какой-либо игры отправляется запрос на специальный адрес с проверкой авторизации, который записывает все необходимое в БД.  
Вся структура приложения находится в папке *app*. Внутри присутсвуют папки *db* (подключение БД), *forms* (формы регистрации/входа), *models* (таблицы БД), *static* (изображения, .css и .js файлы), *templates* (шаблоны .html) и само приложение *main.py*.  
  
**IDE**: PyCharm для бэкенда, VS Code для фронтенда  
**Языки**: HTML, CSS (TailwindCSS, SimpleBar, Google Fonts), JS (Alpine.js), python (flask, [flask-]sqlalchemy, flask-wtf, flask-login)
